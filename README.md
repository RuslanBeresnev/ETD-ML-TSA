# ETD-ML-TSA

- Исходный датасет: https://github.com/zhouhaoyi/ETDataset
- В .ipynb-файле реализовано построение трёх классических ML-моделей для прогнозирования значений временного ряда, описывающего параметры трансформатора:
  1. Загрузка датасета
  2. Создание признаков для моделей
  3. Формирование тренировочной и валидационной выборок
  4. Уменьшение размерности матрицы признаков с помощью PCA
  5. Нормализация полученных признаков
  6. Обучение моделей CatBoostRegressor, LGBMRegressor и XGBRegressor
  7. Сравнение их показателей метрики MSE и показателей baseline-моделей
  8. Построение графиков
